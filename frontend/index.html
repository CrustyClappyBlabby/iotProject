Plant Health Percentage - How is it calculated?

The percentage value for each plant is calculated based on four measured parameters: moisture, temperature, light, and humidity.

  • Optimal value (100%): Parameter is within the plant's optimal range
  • Warning (50%): Parameter is between the optimal range and min/max limits
  • Critical (0%): Parameter is below minimum or above maximum threshold

The overall percentage is the average of all four parameters, and the color indicates the plant's general condition:
